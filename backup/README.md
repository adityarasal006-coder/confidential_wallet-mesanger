# Offline Demo Backups

This directory ensures that if the WiFi completely dies during judging, you can still present flawlessly.

**Action Required:**
Please place the following final files directly into this `backup/` directory before the hackathon ends:

1. `demo-video.mp4` (Your recorded 3-7 minute demo)
2. `architecture.png` (Exported image of your architecture diagram)
3. `pitch.pdf` (Your final presentation slides)
4. Drop your high-res screenshots into the `screenshots/` folder.

If the live demo fails, immediately open `demo-video.mp4` and narrate over it!
